

# 🎤 Interviewer App

A training-focused interview simulation app designed to help users prepare for job interviews while assessing and improving soft skills.

This application provides an interactive environment where users can practice answering interview questions, receive feedback, and refine their communication abilities.

---

## 🚀 Features

* Simulated interview experience
* AI-powered question generation
* Soft skills assessment (e.g. communication, clarity, confidence)
* Real-time interaction via Streamlit
* Easy local setup

---

## 🛠️ Setup Instructions

Follow these steps to run the application locally:

### 1. Clone the Repository

```bash
git clone https://github.com/evolveer/interviewer-app
cd interviewer-app
```

### 2. Create Environment File

```bash
touch .env
```

### 3. Add Your API Key

Open the `.env` file and add your OpenAI API key:

```bash
OPENAI_API_KEY=your_api_key_here
```

You can edit the file using:

```bash
vi .env
```

---

## ▶️ Run the App

Start the Streamlit application with:

```bash
streamlit run interview.py
```

---

## 💡 Purpose

This app is built to:

* Help candidates practice job interviews
* Provide structured feedback on responses
* Improve soft skills critical for professional success
